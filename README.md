# 📁 Project Manager App (React + Tailwind CSS)
**🧩 Description**

Cette application est un gestionnaire de projets moderne développé avec React.js et Tailwind CSS. Elle permet aux utilisateurs de créer, organiser et suivre l’avancement de leurs projets de manière simple et intuitive.

L’objectif principal est d’offrir une interface rapide, responsive et agréable pour gérer des tâches, des équipes et des deadlines.

🚀 Fonctionnalités
✅ Création, modification et suppression de projets
📋 Gestion des tâches par projet
📅 Suivi des deadlines
👥 Attribution de tâches à des membres
📊 Visualisation de l’état d’avancement
🔍 Recherche et filtrage

🛠️ Stack Technique
Frontend : React.js
Styling : Tailwind CSS
